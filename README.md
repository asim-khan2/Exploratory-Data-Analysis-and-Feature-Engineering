# Exploratory Data Analysis and Feature Engineering

## 📌 Project Overview

This repository demonstrates various Exploratory Data Analysis (EDA) and Feature Engineering techniques used in Machine Learning projects. The goal is to understand datasets, identify patterns, handle missing values, detect outliers, and transform data into a suitable format for machine learning models.

## 🎯 Objectives

- Understand the structure of datasets
- Perform data cleaning and preprocessing
- Handle missing values
- Detect and treat outliers
- Analyze feature distributions
- Perform feature encoding
- Apply feature scaling
- Conduct correlation analysis
- Create new features for improved model performance

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Jupyter Notebook

### 4. Feature Selection
